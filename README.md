### üçï Pizzeria App (MEAN Stack)

A full-stack Pizzeria web application built using the MEAN stack (MongoDB, Express.js, Angular, Node.js).

Provides a user-friendly interface to browse pizzas, ingredients, and offers.

Includes a responsive homepage with sections like Our Story, Ingredients, Chefs, and 45-Minute Delivery.

Supports CRUD operations for pizzas and toppings using MongoDB.

Backend APIs developed with Node.js & Express.js.

Frontend built using Angular with component-based architecture.

Uses RESTful services for smooth communication between frontend and backend.

Designed for learning and practicing full-stack MEAN development.
